Profile Management

To update your profile information:

  1. Click on your username in the top navigation bar
  2. Select "Profile" from the dropdown menu
  3. Update your information in the profile form
  4. Click "Save Changes" to update your profile
Note: Some profile fields may be restricted based on your role and permissions.
Task Management
Creating Tasks

Depending on your role, you can create tasks in different ways:

Manual Creation

Use the task creation form to manually create tasks:

  1. Navigate to Tasks → Create Task
  2. Fill in task details (title, description, priority)
  3. Select assignee and due date
  4. Click "Create Task"
AI-Powered Creation

Use AI to convert natural language to tasks:

  1. Go to AI Analysis page
  2. Enter task description in natural language
  3. AI will analyze and suggest task breakdown
  4. Review and confirm task creation

Organization Management

The organization chart displays the hierarchical structure of your team:

  • Director Level: Top management (red nodes)
  • Manager Level: Team leaders (yellow nodes)
  • Employee Level: Team members (green nodes)

AI Features
AI Task Analysis

The AI system helps with:

  • Task Breakdown: Convert natural language to structured tasks
  • Workload Optimization: Balance workload across team members
  • Skill Matching: Match tasks with employee skills
  • Deadline Prediction: Suggest realistic deadlines
Using AI Analysis
  1. Navigate to AI Analysis page
  2. Select analysis type (Workload, Task Optimization, etc.)
  3. Review AI recommendations
  4. Apply suggestions as needed
Tip: The AI system learns from your usage patterns and provides better recommendations over time.
Troubleshooting
Common Issues and Solutions

Problem: Cannot login to the system

Solution:

  • Check your username and password
  • Ensure Caps Lock is not enabled
  • Clear browser cache and cookies
  • Try using incognito/private browsing mode

Problem: Cannot create or assign tasks

Solution:

  • Check your role permissions
  • Ensure assignee is active in the system
  • Verify task details are complete
  • Check for system notifications or errors

Problem: Pages not displaying correctly

Solution:

  • Refresh the page (F5 or Ctrl+R)
  • Clear browser cache
  • Check internet connection
  • Try different browser
